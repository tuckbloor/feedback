JavaScript

The Jquery is very simple

$(document).ready(function() {
   
    $('#comment').bind("keyup", function(e) {
    count=$("#comment").val().length;
    $('#comment').keypress(function(e) {
    var tval = $('#comment').val(),
        tlength = tval.length,
        set = 200,
        remain = parseInt(set - tlength);
    $('#count').text(remain);
    if (remain <= 0 && e.which !== 0 && e.charCode !== 0) {
        $('#comment').val((tval).substring(0, tlength - 1))
    }
})
    });
});

it uses jquery to count 200 characters this can be changed be more or less by changing

set = 200, to what ever the amount of characters you would like to allow
